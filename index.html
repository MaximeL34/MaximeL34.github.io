2025 — 2026 · Healthcare

AI Data Engineer — Ramsay Santé engagement

Talan · Europe's leading private hospital group — parallel AI workstreams sponsored by the client's Data Office, over a ~2.2M inpatient-stays-per-year scope

AI-assisted medical coding

Multi-LLM ICD-10 coding pipeline: benchmark of GPT, Gemini, Mistral, Grok and Claude with weighted hierarchical scoring, RAG (FAISS), NER pseudonymization (Docker API) and a React validation interface for medical coders. Pilot: 100% accuracy on the primary diagnosis vs 63% human baseline. LangGraph multi-agent architecture (coder → QA → arbitration) designed for scale-up.

Billing rejection control (health insurance)

Hybrid LLM + deterministic Python pipeline detecting insurance billing rejections on an 81-invoice benchmark. Diagnosed a GPT model-version drift on Azure (96% → 59-72%), analyzed false-positive patterns, and got the hybrid architecture adopted: deterministic rules for structured rejection reasons, LLM for semantic ones. Built the 24-code rejection reference base.

Predictive ML for billing rejections

Predictive detection of invoices at risk of rejection: XGBoost chosen over LLM fine-tuning for auditability, cost and reproducibility, with SHAP explainability. Relational data from the replicated invoicing database, Azure ADLS Gen2 target, batch scoring and FastAPI exposure.

Hospital funding engine (French DRG / PMSI)

Complete Python DRG grouping engine (2026 national rules): positional record parsing, classifying procedures, severity levels, tariff valuation with outlier payments and ICU supplements — plus an under-coding detection framework scoring upgradeable stays for revenue recovery. Proof that I can absorb a country-specific regulatory system and re-implement it from official specs.

Azure platform & automated intelligence

Automated public-funding watch system, architecture validated with the client's Azure architect: PostgreSQL + pgvector (HNSW, 3072-dim embeddings), Bing Grounding, Key Vault CSI, Helm + ArgoCD deployment on AKS. Framing documents delivered in English and French.
